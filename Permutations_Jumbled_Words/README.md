# 🎮 Permutations Jumbled Words Game

## 📌 Overview

This is a **two-player word guessing game** based on permutations (jumbled words).

One player is given a **jumbled version of a word**, and they must guess the correct English word. The game alternates turns between two players and keeps track of their scores.

---

## 🎯 Objective

* Improve vocabulary and thinking skills
* Understand permutations (jumbled words)
* Practice Python concepts through a fun game

---

## 🧩 Game Rules

* Two players participate in the game
* A random word is selected and jumbled
* Players take turns to guess the correct word
* If the guess is correct → player earns 1 point
* If incorrect → correct word is revealed
* Game continues until a player chooses to quit

---

## 🚀 Features

* 🎲 Random word selection
* 🔀 Word jumbling using permutations
* 👥 Two-player turn-based system
* 📊 Score tracking
* 🏆 Winner announcement at the end
